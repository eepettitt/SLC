getFolders - copies the names of all folders in the parent folder of the script file to the clipboard

Restart Explorer - closes and reopens Windows Explorer

CheckFlat - prompts user for a part number or partial part number and lists the dxf files in the Z drive that match
the FileTransfers folder must be mapped to Z:\ on the local computer; use * as a wildcard for partial part numbers

kill wscript - closes instances of Windows Script Host

MoveFolders - moves all folders starting with "P" or "0" in the parent folder to their -Scott part folder locations; copy script to parent folder to use
